# CodeLeap Test

## Instructions

### Install dependencies
```
npm install
```

### Setup env variables

Here you need to configure the VITE_APIURL environment variable in order to run it properly. If you don't, you are not going to be able to execute the CRUD functions for posts.
- VITE_APIURL

### Run
```
npm run dev
```

## Libraries
- Axios
- Dayjs
- Redux
- Styled components
